DÚVIDAS

Perguntas frequentes

Jogadores Java e Bedrock podem jogar juntos?+

Sim. O Log Horizon foi preparado como um servidor cross-platform, permitindo que as duas edições compartilhem o mesmo mundo.

O aplicativo é obrigatório?+

O aplicativo é necessário apenas para jogadores que desejam usar o sistema de voz compatível no Android. O acesso normal ao servidor continua separado.

O aplicativo é seguro?+

Baixe sempre por este site oficial. O APK distribuído aqui é a versão release assinada do projeto Log Horizon Voice.

Onde recebo suporte?+

O link oficial da comunidade será adicionado nesta página para dúvidas, suporte e anúncios.
